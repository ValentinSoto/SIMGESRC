# SIMGESRC - Simulador de Gestión de Recursos del Sistema Operativo

**SIMGESRC** es un simulador educativo escrito en Python diseñado para ilustrar los conceptos fundamentales de los sistemas operativos: arquitectura del kernel, planificación de procesos, gestión de memoria y sincronización.

## 📁 Estructura del Proyecto

- `docs/`: Documentación teórica y guías paso a paso (.md e imágenes).
- `src/`: Código fuente del simulador (Kernel, Scheduler, MMU, Sync).
